# Stockora – Stock Market Explorer

## Project Overview

Stockora is a web application that allows users to search and explore stock market data in a simple and interactive way. The application fetches stock information from a public API and displays it dynamically on the webpage.

The main goal of this project is to demonstrate understanding of JavaScript concepts such as API integration, asynchronous programming, and building dynamic user interfaces.

---

## Features (Planned)

* Search stocks by name or symbol
* Display stock price, change, and percentage change
* Sort stocks based on price or performance
* Filter stocks (gainers and losers)
* Add and remove stocks from favorites using local storage
* Toggle between light mode and dark mode (optional)

---

## API Used

This project uses the Alpha Vantage API to fetch stock market data.

API Link: https://www.alphavantage.co/

---

## Technologies Used

* HTML
* CSS (Vanilla CSS for styling)

## How to Run the Project

1. Clone the repository to your local machine

2. Open the project folder

3. Open the `index.html` file in any web browser

4. The application will run locally in your browser
